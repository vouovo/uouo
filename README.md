# 📘 四六级英语 AI 答疑助手

> 基于阿里云百炼通义千问（Qwen）大模型，纯前端、零后端的大学生英语学习工具。  
> 支持文字提问、图文混合提问，内置七大专业答疑场景，即开即用，适配电脑与手机。

🔗 **在线体验**：https://vouovo.github.io/uouo

---

## ✨ 项目简介

本项目是一款专为大学生四六级英语备考设计的智能答疑助手。它直接调用阿里云百炼平台的通义千问大模型（支持 `qwen-plus`、`qwen3.7-plus`、`qwen-vl-plus`），无需自建后端，所有代码在浏览器中运行。  
您只需填入自己的 API Key，即可享受专业、即时、场景化的英语学习辅助服务。

---

## 🎯 功能说明

### 核心功能
- **智能问答**：输入英语学习问题，AI 实时生成专业回答。
- **七大场景自动切换**（每个场景配有专属系统提示词）：
  - 📖 单词查询 – 音标、词根、变形、高频搭配、记忆技巧
  - 🔍 语法长难句 – 成分拆解、从句解析、语法纠错、翻译、润色
  - ✍️ 四六级专项答疑 – 阅读/听力/写作技巧、考点、避坑指南
  - 📝 四六级作文批改 – 15 分制评分、语法/拼写纠错、优化范文
  - 🗣️ 口语指导 – 地道句式、考试模板、中式英语纠正
  - 🌐 中英互译 – 双向翻译、润色、文化知识点
  - 💬 通用英语问答 – 无限制解答各类英语问题

### 交互体验
- 支持上传 **JPG / PNG / GIF** 图片（≤5MB），模拟 OCR 识别试卷、作文、错题。
- 图片实时预览、一键移除。
- 输入框支持 `Enter` 快速发送，`Shift+Enter` 换行。
- 对话气泡区分用户与 AI，界面清爽。

### 数据管理
- **本地存储**（localStorage）：刷新页面对话不丢失。
- **历史会话列表**：侧边栏展示全部对话记录，点击一键恢复。
- **一键清空**（带确认弹窗）与 **导出 TXT**（自动命名含时间日期）。

### 个性化配置
- 自定义 **系统提示词**，自由设定 AI 角色与回答风格。
- 选择不同的 Qwen 模型（`qwen-plus` / `qwen3.7-plus` / `qwen-vl-plus`）。
- API Key 由用户手动输入，**绝不硬编码**。

## 🚀 运行方法

1. **下载代码**  
   将 `index.html` 文件保存到本地（或克隆整个仓库）。

2. **获取 API Key**  
   - 登录 [阿里云百炼控制台](https://bailian.console.aliyun.com/)。
   - 创建应用并获取 API Key（格式为 `sk-xxx`）。

3. **打开页面**  
   直接用浏览器（Chrome / Edge / Firefox 等）打开 `index.html` 即可运行。

4. **配置密钥**  
   在页面左侧侧边栏的“模型配置”区域，粘贴您的 API Key，选择模型，然后开始提问。

> ⚠️ 注意：首次使用需手动输入 API Key，之后会自动保存在浏览器中（仅保存在本地）。

## 📦 部署方法

本项目为纯静态页面，可部署到任何支持托管 HTML 的平台：

### 方式一：GitHub Pages（推荐）
1. 将 `index.html` 上传至您的 GitHub 仓库。
2. 进入仓库 `Settings` → `Pages`，选择分支并保存。
3. 等待几分钟，访问 `https://<用户名>.github.io/<仓库名>` 即可。

无需任何后端环境，上传单个 HTML 文件即可工作。

## ⚠️ 注意事项

1. **API Key 安全性**  
   - 您的 API Key 仅保存在浏览器 `localStorage` 中，**不会**上传到任何第三方服务器。
   - 建议在公共电脑上使用完毕后手动清除浏览器数据或点击“清空全部历史”时一并清除 Key（需手动从设置中删除）。
   - 切勿将包含真实 API Key 的代码提交到公开仓库。

2. **模型选择**  
   - 使用 `qwen-vl-plus` 可支持图片识别，但需确保您的 API Key 有相应权限。
   - 纯文本模型（`qwen-plus` / `qwen3.7-plus`）无法解析图片内容，上传图片时会被转为文本描述提示。

3. **浏览器兼容**  
   建议使用最新版 Chrome / Edge / Firefox，以保证 `localStorage` 和 ES6 语法正常工作。

4. **网络要求**  
   需要能正常访问阿里云百炼 API 端点（`dashscope.aliyuncs.com`），国内网络通常无问题。

5. **图片大小**  
   单张图片限制 **5MB**，超出会提示上传失败。

## 🔐 API Key 安全说明

- **存储位置**：`localStorage`（浏览器本地），键名为 `eng_ai_settings`。
- **传输方式**：仅在与阿里云百炼 API 通信时通过 HTTPS 请求头 `Authorization: Bearer <key>` 传递。
- **风险提示**：由于是纯前端应用，API Key 存在被恶意浏览器插件窃取的理论风险。建议：
  - 仅在个人设备上使用。
  - 定期在百炼控制台重置 API Key。
  - 避免在公开或共享电脑上长期保存 Key。
